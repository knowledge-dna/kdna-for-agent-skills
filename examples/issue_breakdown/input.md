# Example: Issue Breakdown

## Input

```
User: "Take this PRD and break it into GitHub issues."

PRD: Team Collaboration Feature
- Users can create teams
- Teams have members with roles (owner, admin, member)
- Team members can share projects
- Activity feed shows recent team actions
- Email notifications for team events
```

## KDNA Domains Activated

- **requirement_alignment** — primary (the PRD needs alignment verification before decomposition)
- **test_driven_development** — secondary (each issue should be a vertical slice)

Note: `issue_decomposition` domain is planned and would be the primary domain here.
